# monkey_climb
Calculates the minimum number of jumps required for a monkey to reach the top of a pole considering climb distance and slip distance.
# Monkey Climb Problem

## Problem Statement
The task is to calculate the number of jumps required by a monkey to reach the top of a pole.

The monkey is trying to climb a pole of height H meters.

- In every jump, the monkey climbs X meters.
- Due to the slippery surface of the pole, the monkey slips down Y meters after each jump.
- Once the monkey reaches or crosses the top, it will not slip again.

Write a program to find the minimum number of jumps required for the monkey to reach the top.

---

## Input Format
Three integers:
H → Height of pole  
X → Climb distance per jump  
Y → Slip distance after jump  

---

## Output Format
Print the minimum number of jumps required.
